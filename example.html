Cardify

Plug-in para imágenes

¿Cómo lo instalo?

Descargar el siguiente script y enlazar en su archivo html las rutas:

<script src="http://code.jquery.com/jquery-3.3.1.min.js">

<script src="cardify.js">

¿Cómo lo uso?

Utilizar el siguiente código para aplicar 'Cardify' a nuestras imágenes:

$(document).ready(()=>{ $('img').cardify(); });
